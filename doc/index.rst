Installation
------------

PlyData can be installed in a handful of ways.

1. Official release *(Recommended)*

   .. code-block:: console

       $ pip install plydata

   If you don't have `pip`_ installed, this `Python installation guide`_
   can guide you through the process.


2. Development sources

   .. code-block:: console

       $ pip install git+https://github.com/has2k1/plydata.git

   Or

   .. code-block:: console

       $ git clone https://github.com/has2k1/plydata.git
       $ cd plydata
       $ python setup.py install

   Or

   .. code-block:: console

       $ curl -OL https://github.com/has2k1/plydata/archive/master.zip
       $ unzip master
       $ cd plydata-master
       $ python setup.py install
